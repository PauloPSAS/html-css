Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em negrito usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em itálico usando a tag <em> (semântica)

Texto marcado

Podemos também criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído para indicar que ele pode ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. tag usada é: <ins> (semântica)

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x10 * 2.

Texto subscrito

Para inserir coisas do tipo: H2O
